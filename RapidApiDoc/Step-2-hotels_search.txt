Query Params
entityId*
String
•	You need to call the API multiple times until the item completionPercentage=100 (data->status->completionPercentage)
•	entityId can be retrieved from /hotels/auto-complete endpoint(data->entityId)
•	Ex: 27537542
checkin*
Date (yyyy-mm-dd)
•	Check-in date
•	Required: true
•	Ex: 2024-01-21
•	Format: YYYY-MM-DD
checkout*
Date (yyyy-mm-dd)
•	Check-out date
•	Required: true
•	Ex: 2024-01-22
•	Format: YYYY-MM-DD
resultsPerPage(optional)
Number
•	Total number of record per api call
•	Ex: 2
•	Default value: 25
page(optional)
Number

•	Ex: 2
•	Default value: 1
rooms(optional)
Number
•	Rooms
•	Ex: 6
•	Default value: 1
adults(optional)
Number

childrenAges(optional)
String

String

locale(optional)
String

currency(optional)

sorting(optional)
String

•	Ex: -price

•	-price: Price (high to low)
•	price: Price (low to high)
•	distance: Distance from downtown
•	-rating: Guest rating
•	-stars: Stars (5 to 1)
•	stars: Stars (1 to 5)
priceType(optional)
String

minPrice(optional)

maxPrice(optional)
String

•	Ex: 500 or 500.4

String
